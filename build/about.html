ASDF

ASDF.cool is ¬

  1. A simple way to manage your bookmarks.
  2. All about displaying your links how you prefer through simple design options.
  3. A personal tool I decided to make available. I use it to link to sites I frequently visit in lieu of the favorites bar in my browser. I don't like all that extra browser chrome visible, and like looking at nice type, so, why not.
  4. Better suited to desktop, altho there is nothing keeping you from using mobile.

  6. If you're looking for more functionality scope Pinboard.

Design options ¬

  1. Font can be set to any font locally installed on your machine. Default is Bau Medium as a webfont.
  2. Columns toggles the display of your links in columns.
  3. Column width adjusts the min size of your columns. These are fluid.
  4. Make it day/night sets the tone of the page. If you really want the ability for custom colors or w/e feel free to request but it wasn't a priority.
  5. Type size defines the size of your links.
  6. Margin size adjusts the space between each of your links. This value is a multpile of the type size, so everything stays in proportion as you finesse.
  7. Import/export enables you to transfer and backup your links. Because everything is stored as an object in a local DB it isn't possible to sync, but this is sort of handy. Local storage can also be a little finnicky, so I suggest exporting if you have a bunch of links added.
  8. Reset links dumps your shit.
  9. About brought you here. Hello.

Your data is ¬

  1. Stored in a local database in your browser.
  2. Not syncable between machines.
  3. Exportable/importable through the Settings panel.
